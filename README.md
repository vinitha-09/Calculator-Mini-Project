# Calculator Mini Project

## Description

A simple web-based calculator developed using HTML, CSS, and JavaScript.

## Objective

To perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear Screen

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Files

* index.html
* style.css
* script.js

## How to Run

1. Open index.html in a web browser.
2. Use the calculator interface.
3. Perform arithmetic calculations.

## Conclusion

This project demonstrates the use of HTML, CSS, and JavaScript to build an interactive calculator application.
